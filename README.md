## 🎯 Aperçu du Projet

Cette application offre une interface interactive pour explorer des cartes du monde avec différents styles de rendu. Elle permet aux utilisateurs de visualiser des données géospatiales, d'ajouter des marqueurs personnalisés, et d'explorer différentes régions de la planète avec des outils de navigation avancés.

### Objectifs
- Créer une interface intuitive pour l'exploration géospatiale
- Intégrer différents styles de cartes (OpenStreetMap, Stamen, CartoDB)
- Permettre l'ajout et la gestion de marqueurs personnalisés
- Offrir des outils de mesure et de navigation

## ✨ Fonctionnalités

### 🗺️ Cartographie Interactive
- **Styles de cartes multiples** : OpenStreetMap, Stamen Terrain, Stamen Toner, CartoDB
- **Navigation fluide** : Zoom, panoramique, centrage automatique
- **Contrôles avancés** : Couches, mesure de distance, géolocalisation

### 📍 Gestion des Marqueurs
- **Marqueurs prédéfinis** : Lieux emblématiques avec informations détaillées
- **Marqueurs personnalisés** : Ajout de nouveaux points d'intérêt
- **Popups informatifs** : Coordonnées, descriptions, emojis

### 🔧 Outils Intégrés
- **Mesure de distance** : Calcul de distances entre points
- **Mini-carte** : Vue d'ensemble de la position
- **Mode plein écran** : Immersion totale
- **Géolocalisation** : Localisation automatique

### 💾 Export et Sauvegarde
- **Export HTML** : Sauvegarde des cartes générées
- **Ouverture navigateur** : Visualisation directe dans le browser
- **Fichiers temporaires** : Gestion automatique des ressources
